# public-pictures

寒枫的公开图片库，一个按主题分类维护的个人图片资源仓库。

这个仓库主要收录学习笔记、技术整理、知识图解和部分兴趣向内容配图，可直接用于：

- GitHub / Markdown 文档插图
- 博客、笔记、知识库配图
- 个人项目中的静态图片引用
- 学习资料中的流程图、结构图、示意图展示

## 仓库特点

- 按专题分类，目录清晰，便于检索和复用
- 当前共 `29` 个主题目录
- 当前共收录 `550` 张图片资源
- 图片格式以 `webp` 为主，同时包含少量 `svg` 与 `gif`
- 文件名语义化，适合作为个人图床和知识型素材仓库
